# 水光镜 
手机后置 9:16 水下水光互动 ASMR 拍摄工具。

## 使用方式

1. 用 HTTPS 地址打开页面。
2. 点击“开启镜头”并允许摄像头权限。
3. 默认进入后置镜头、9:16 竖屏安全框和手机拍摄 HUD。
4. 底部按钮可录制、拍照、切换镜头、切换预设和展开参数面板。

## 隐私

摄像头画面、麦克风分析、水声合成和录制都在本地浏览器内完成，不上传素材。

## 手机兼容

- iOS Safari / Android Chrome 建议使用最新版。
- 摄像头必须运行在 HTTPS 页面，GitHub Pages 地址可直接使用。
- 如果浏览器不支持网页内录制，可以使用系统录屏。
